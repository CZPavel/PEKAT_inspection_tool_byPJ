# Release 3.6

Datum vydání: 2026-02-23

## Přehled

Release 3.6 sjednocuje Sound camera funkce, GUI layout optimalizace a release/build vrstvu:

1. Povýšení verze na `3.6.0`.
2. Aktualizace názvů v GUI a dokumentaci na `V03.6`.
3. Build EXE s novou ikonou a release dist názvem `PEKAT_Inspection_tool_by_PJ_V03_6`.
4. Aktualizace user/technical dokumentace a doplnění mapy návazností.

## Co je nové

1. GUI:
   - title `PEKAT Inspection Tool V03.6`
   - pokračující optimalizace rozložení a práce se Sound camera kartou
2. Sound camera:
   - integrované přístupy `Payload`, `Lissajous`, `Classic`
   - Classic styly `CLASSIC`, `FUSE7`, `FUSE4_base`
3. Build:
   - PyInstaller build s ikonou `resources/branding/pektool-ico.ico`
   - sjednocené metadata release verze `3.6.0`

## Artefakty

Po build procesu:

1. `dist/PEKAT_Inspection_tool_by_PJ_V03_6/pektool-gui.exe`
2. `dist/PEKAT_Inspection_tool_by_PJ_V03_6/pektool.exe`

Distribuční ZIP:

1. `PEKAT_Inspection_tool_by_PJ_V03_6_win64.zip`
